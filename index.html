About

The WUnderground plugin allows users to create Indigo Home Control Server devices which provide a facility to create individual weather devices that will be linked to Weather Underground.

Installation

Installation is easy. Download and extract the Indigo Plugin file onto your Indigo server machine. Double-click the plugin file and follow the prompts. The plugin contains functionality to create an unlimited number of weather devices, but there can be a limit to the number of times you can download Weather Underground data depending on your plan type.

Devices

The WUnderground Plugin supports several device types.

  • Current Weather Conditions with short-term (day/night) forecast
  • 10-Day Weather Forecast
  • 24-Hour Weather Forecast
  • Astronomy Information
  • Weather Almanac
  • Tide Forecast
  • Image Downloader (for satellite imagery)

Actions

There is one action defined within the plugin.

  • Refresh Weather Data

Menu Items

In addition to the standard plugin menu items, the WUnderground plugin also provides:

  • Enable all plugin devices
  • Disable all plugin devices
  • Refresh Data Now
  • Write Weather Data to File
  • Check for Plugin Updates
  • Toggle Debugging
